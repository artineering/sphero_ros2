Quick Reference: Available Tasks & Conditions
Entry Condition Types:
  • always - Always allow entry (no params needed)
  • timer - Wait for duration: {"type": "timer", "params": {"duration": 5.0}}
  • topic_value - Check sensor value: {"type": "topic_value", "params": {"key": "battery_percentage", "operator": "<", "value": 20}}
Operators: ==, !=, >, <, >=, <=
Available sensor keys: pitch, roll, yaw, accel_x, accel_y, accel_z, gyro_x, gyro_y, gyro_z, x, y, velocity_x, velocity_y, battery_percentage
Transition Condition Types (NEW!):
  • auto - Uses destination state's entry condition (default)
  • timer - Time-based: {"type": "timer", "duration": 3.0}
  • topic_value - Topic field comparison: {"type": "topic_value", "topic": "/cmd", "msg_type": "std_msgs/String", "field_path": "data", "operator": "==", "value": "go"}
  • topic_message - Any message received: {"type": "topic_message", "topic": "/trigger", "msg_type": "std_msgs/Bool", "timeout": 1.0}
Add "condition": {...} to any transition to use topic-based triggering!
Task Types (use in "tasks" array):
  • set_led - Set LED color: {"type": "set_led", "params": {"color": "red", "led": "main"}}
  • roll - Move robot: {"type": "roll", "params": {"heading": 0, "speed": 100, "duration": 2.0}}
  • stop - Stop robot: {"type": "stop", "params": {}}
  • matrix - Show LED pattern: {"type": "matrix", "params": {"pattern": "smile", "red": 255, "green": 255, "blue": 255}}
  • heading - Set heading: {"type": "heading", "params": {"heading": 90}}
  • speed - Set speed: {"type": "speed", "params": {"speed": 128, "duration": 3.0}}
LED types: main, front, back | Colors: red, green, blue, yellow, cyan, magenta, white, orange, purple, pink, off
